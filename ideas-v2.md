# DEO Conseil — Refonte Design V2 : Couleur, Innovation, Impact

## Contexte
Le client veut sortir du noir/or corporate. Il veut de la couleur, de l'innovation, des jeux d'accordéon pour les services, et un design qui attire l'œil. Le site doit rester prestigieux mais devenir vibrant et mémorable.

## Idée 1 : "Electric Gradient Flow"
<probability>0.07</probability>

**Design Movement** : Neo-Brutalism + Gradient Maximalism — inspiré par Stripe, Linear, et les sites tech les plus audacieux.

**Core Principles** :
1. Gradients vivants comme élément structurel (pas décoratif)
2. Typographie massive et expressive avec des contrastes de taille extrêmes
3. Cartes interactives avec hover states spectaculaires
4. Mouvement continu — le site "respire"

**Color Philosophy** :
- Fond : Noir profond (#050505) pour faire exploser les couleurs
- Gradient principal : Violet électrique (#7C3AED) → Bleu cyan (#06B6D4) → Vert émeraude (#10B981)
- Accent chaud : Orange corail (#F97316) pour les CTA
- Texte : Blanc pur (#FFFFFF) et gris clair
- Le gradient symbolise la transformation : du violet (vision) au vert (croissance)

**Layout Paradigm** : Sections full-width avec des blobs de gradient animés en arrière-plan. Bento grid pour les services. Scroll horizontal pour les témoignages.

**Signature Elements** :
1. Blobs de gradient animés qui flottent en background
2. Texte avec gradient clip — les titres "brillent"
3. Cartes avec border-gradient et glow effect au hover

**Interaction Philosophy** : Chaque interaction déclenche un micro-feedback visuel. Les cartes se soulèvent, les gradients bougent au survol, les accordéons s'ouvrent avec une animation fluide.

**Animation** : Blobs qui se déplacent lentement, parallax subtil, entrées staggered, accordéons avec spring physics.

**Typography** : Clash Display (titres — géométrique, bold, moderne) + Inter (corps — lisible, neutre)
</response>

## Idée 2 : "Bauhaus Meets Africa"
<probability>0.05</probability>

**Design Movement** : Neo-Bauhaus + Afrofuturisme — formes géométriques audacieuses, couleurs terreuses vibrantes, motifs africains abstraits.

**Core Principles** :
1. Géométrie forte : cercles, triangles, lignes diagonales comme système visuel
2. Couleurs terreuses mais vibrantes — terre, safran, indigo, terracotta
3. Asymétrie intentionnelle — rien n'est centré par défaut
4. Hommage subtil à l'identité marocaine/africaine

**Color Philosophy** :
- Fond : Bleu nuit profond (#0F172A)
- Primaire : Safran doré (#F59E0B)
- Secondaire : Terracotta (#DC6843)
- Accent : Indigo (#4F46E5)
- Vert olive (#84CC16) pour les éléments de succès
- Les couleurs racontent l'Afrique : terre, soleil, ciel, végétation

**Layout** : Grille asymétrique avec des éléments qui débordent. Sections avec des découpes diagonales. Cards empilées en cascade.

**Signature Elements** :
1. Formes géométriques flottantes (cercles, triangles) en arrière-plan
2. Bordures colorées épaisses sur les cartes
3. Motifs géométriques subtils inspirés du zellige

**Typography** : Syne (titres — géométrique, africain-futuriste) + DM Sans (corps)
</response>

## Idée 3 : "Vivid Corporate Rebellion"
<probability>0.08</probability>

**Design Movement** : Corporate Memphis inversé + Swiss Design coloré — prendre les codes corporate et les exploser avec de la couleur.

**Core Principles** :
1. Couleurs vives sur fond sombre — chaque section a sa propre identité colorée
2. Typographie XXL — les chiffres et titres dominent visuellement
3. Interactivité poussée — accordéons, tabs, hover reveals, curseur custom
4. Sections qui se distinguent par des palettes différentes mais harmonieuses

**Color Philosophy** :
- Fond principal : Charcoal très sombre (#0C0C0C)
- Bleu électrique (#2563EB) — section Hero et Vision
- Violet profond (#7C3AED) — section Services
- Émeraude (#059669) — section Compétences
- Orange vif (#EA580C) — section Fondateurs
- Rose magenta (#DB2777) — section Fabrik RH
- Chaque section a un "glow" de sa couleur dominante

**Layout** : Bento grid pour les services avec accordéons intégrés. Sections full-bleed avec transitions de couleur. Cards avec des bordures colorées épaisses à gauche.

**Signature Elements** :
1. Chaque section a un "color glow" — une lueur subtile de sa couleur dominante
2. Numéros géants (01, 02, 03...) en arrière-plan des sections
3. Lignes de séparation colorées et animées entre les sections

**Interaction** : Accordéons avec animation spring, hover qui révèle du contenu caché, parallax sur les images, cursor trail coloré.

**Animation** : Entrées staggered avec des directions variées, color morphing entre sections, compteurs animés, text reveal lettre par lettre pour les titres.

**Typography** : Plus Jakarta Sans (titres — moderne, géométrique, bold) + Inter (corps)
</response>

---

## Choix : Idée 3 — "Vivid Corporate Rebellion"

**Pourquoi** : C'est la direction qui répond le mieux à la demande — couleurs vibrantes, innovation, accordéons pour les services, et un design qui attire. Chaque section a sa propre identité colorée, ce qui crée un parcours visuel dynamique. Le fond sombre fait exploser les couleurs. Les accordéons interactifs pour les services sont au cœur du concept.
